How To Create A Texture Pack


Step 1
Download the Blank Texture Pack HERE

Step 2
Login to Scratch

Step 3
Create a New Project

Step 4
Go to File > Load from your computer and Select the Blank Texture Pack.

Step 5
Edit the Costumes and/or the "Item Names" List

Step 6
Save the Project ID (The number found at the end of the link)
